# Dataset Herbal Leaves

## About The Project

Project ini merupakan kumpulan dataset daun herbal. Dengan 2 (dua) label, dengan masing-masing label terdiri atas 50 image (total 100 image).

- Daun Lidah Buaya.
- Daun Kemangi.

## Dataset Specifications

Dataset ini memiliki spesifikasi sebagai berikut :

- Image daun diambil menggunakan kamera smartphone
- Background Putih
- Format .jpg
- Dimensi 1600 X 1200 pixel
